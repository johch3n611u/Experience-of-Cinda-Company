## 1

軟體測試：在規定的條件下對程式進行操作，以發現程式錯誤，衡量軟體品質，並對其是否能滿足設計要求進行評估的過程。

有許多方法

### 進程分類

Alpha 驗證測試 : 通常是在開發單位由開發人員與測試的測試人員，以類比或實際操作性的方式進行驗證測試。

Beta 確認測試 ( Close、Open ) : 由公眾參與的測試，真實的環境中以實際的資料來執行測試，以確認效能，系統執行有效率，系統復原與備份作業正常。
